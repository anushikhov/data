## Outer Joins, Cross Joins and Self Joins  

**LEFT JOIN will return all records in the left table, and those records in the right table that match on the joining field provided.**    
  

**left table**  

| id            | left\_val     |
| ------------- |:-------------:|
| 1             | L1            |
| 2             | L2            |
| 3             | L3            |  
| 4             | L4            |  

**right table**  

| id    | right\_val |
|-------|:----------:|
| 1     | R1         |
| 4     | R2         |
| 5     | R3         |
| 6     | R4         |
